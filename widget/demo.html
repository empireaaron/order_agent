🤖 TicketBot AI智能客服演示

关于

这是一个基于 LangGraph + LLM 的智能客服系统演示。点击右下角的 💬 聊天按钮开始对话。

🔍 意图识别 - 自动理解用户需求
📋 智能工单 - 自动提取标题、优先级、分类
💡 知识库问答 - RAG检索增强
📊 工单统计 - 实时数据查询

测试用例

点击以下测试语句快速体验，或在聊天框中输入：

[创建工单]我的订单无法支付，一直显示错误
[查询工单]查询我的工单状态
[统计]我的工单统计
[知识库]如何使用退货功能
[通用]你好

使用方法

在您的网站中嵌入以下代码：

<script src="ticket-widget.js" defer></script>
<script>
  TicketWidget.init({
    apiUrl: 'http://localhost:8000/api/v1',
    websocketUrl: 'ws://localhost:8000/ws',
    theme: 'blue',
    showBubble: true
  });
</script>
